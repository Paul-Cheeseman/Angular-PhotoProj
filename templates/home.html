Site Purpose:

The purpose of this website is to show some of the front-end skills I have been developing through the Full Stack Web Development course run by the Code Institute. It has the dual purpose of covering the examination requirements while also showing prospective employers what I am capable of.

Site Content:

The main content is on the media page is based around a hobby of mine, which is landscape photography. It’s at an amateur level but hopefully some nice pictures none the less. There is a basic bio of my career and academic achievements in the profile page and a dummy contact me form in the contact page. The form will validate input, but it currently won’t send the information anywhere, so please don’t try to contact me though it.

Site Aim:

My main aim has been to try and make the site as scaleable as possible. I wanted the site to look good and be useable through all possible viewing devices. Whilst there has only been a short time scale for me to complete this site (as I wish to progress with the rest of the course) I feel that I have achieved that main aim. I also wanted to implement some pure JavaScript into site and have achieved that with the the picture viewer and the additional validation I have put in for the contact me forms date field. I also wanted this site to be user friendly and believe that I have achieved that by implementing defensive programming in both the picture viewer and form input.

Technology Used:

For layout I have used bootstrap, although it is not working as default on this site because I have deliberately overwritten some of the default settings to enable better scalability, an example of that is the scaling of text to suit the device being used. Also, while bootstrap also has default plug-ins/code, I wanted to illustrate some programming skills and as such did not use the bootstrap carousel, I created my own initially based on one from w3schools, but have heavily adapted it, as can be seen from the source JavaScript.
